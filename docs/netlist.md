# Netlist Import / Export

WireScript supports exporting schematics to industry-standard netlist formats and importing netlists back into the `WireScriptDb` representation.

---

## Supported Formats

| Format | ID | Extension | Description |
|---|---|---|---|
| SPICE | `'spice'` | `.net`, `.cir`, `.sp` | Standard SPICE netlist ŌĆö compatible with LTspice, ngspice, KiCad, etc. |
| WireScript CSV | `'ws-csv'` | `.csv` | WireScript's own CSV netlist ŌĆö easy to read, edit, and process |

---

## `exportNetlist(db, options?)` ŌĆö DB ŌåÆ Netlist

Converts a `WireScriptDb` to a netlist string.

```ts
import { Circuit, DC, R, LED, GND, RED, compileDslToDb, exportNetlist } from '@ssevindikx/wirescript';

const circuit = Circuit('LED Driver', DC(5), R(330), LED(RED), GND());
const db = compileDslToDb(circuit);

// SPICE format (default)
const spice = exportNetlist(db);

// WireScript CSV format
const csv = exportNetlist(db, { format: 'ws-csv' });
```

### Options

```ts
interface NetlistExportOptions {
  format?:   'spice' | 'ws-csv';  // Output format (default: 'spice')
  title?:    string;               // Title comment in SPICE output (default: schematic name)
  spiceEnd?: boolean;              // Include .end directive (default: true, SPICE only)
}
```

### SPICE output example

```spice
* LED Driver
* Generated by WireScript db2netlist

V1 N1 0 5
R1 N1 N2 330
D1 N2 0 red

* Ground nets: 0

.end
```

### WireScript CSV output example

```csv
refdes,type,pin,net,value,model
V1,voltage_source,positive,N1,5,
V1,voltage_source,negative,0,5,
R1,resistor,1,N1,330,
R1,resistor,2,N2,330,
D1,led,anode,N2,0,red
D1,led,cathode,0,0,red
GND1,ground,gnd,0,,
```

## `importNetlist(src, options?)` ŌĆö Netlist ŌåÆ DB

Parses a netlist string and returns a `WireScriptDb`.

```ts
import { importNetlist } from '@ssevindikx/wirescript';
import { promises as fs } from 'fs';

// From a SPICE file
const spiceSrc = await fs.readFile('circuit.net', 'utf-8');
const db = importNetlist(spiceSrc);

// From a CSV file
const csvSrc = await fs.readFile('circuit.csv', 'utf-8');
const db2 = importNetlist(csvSrc, { format: 'ws-csv' });
```

Format is **auto-detected** by default:
- First line matches `refdes,` ŌåÆ `ws-csv`
- Otherwise ŌåÆ `spice`

### Options

```ts
interface NetlistImportOptions {
  format?: 'spice' | 'ws-csv';  // Input format (default: auto-detected)
  name?:   string;               // Schematic name (default: parsed from SPICE title or 'imported')
}
```

## Aliases

All four alias pairs work identically:

```ts
import {
  exportNetlist, dbToNetlist, db2netlist,  // DB ŌåÆ netlist
  importNetlist, netlistToDb, netlist2db,  // netlist ŌåÆ DB
} from '@ssevindikx/wirescript';
```

## SPICE reference designator mapping

| WireScript type | SPICE prefix | Example |
|---|---|---|
| `resistor` | `R` | `R1` |
| `capacitor` | `C` | `C1` |
| `inductor` | `L` | `L1` |
| `diode`, `led` | `D` | `D1` |
| `voltage_source` | `V` | `V1` |
| `current_source` | `I` | `I1` |
| `bjt`, `npn`, `pnp` | `Q` | `Q1` |
| `mosfet`, `nmos`, `pmos` | `M` | `M1` |
| `opamp` | `U` | `U1` |

### SI value suffixes (import)

| Suffix | Multiplier | Example |
|---|---|---|
| `T` | ├Ś 10┬╣┬▓ | `1T` = 1 T╬® |
| `G` | ├Ś 10Ōü╣ | `1G` = 1 GHz |
| `MEG` | ├Ś 10ŌüČ | `2.2MEG` = 2.2 M╬® |
| `K` | ├Ś 10┬│ | `10K` = 10 k╬® |
| `M` | ├Ś 10Ōü╗┬│ | `100M` = 100 mA |
| `U` | ├Ś 10Ōü╗ŌüČ | `47U` = 47 ┬ĄF |
| `N` | ├Ś 10Ōü╗Ōü╣ | `10N` = 10 nH |
| `P` | ├Ś 10Ōü╗┬╣┬▓ | `100P` = 100 pF |
| `F` | ├Ś 10Ōü╗┬╣ŌüĄ | |

> **Note**: `G` is checked after `MEG` to ensure `2.2MEG` is not misread as `2.2M + 'EG'`.

## Round-trip example

```ts
import {
  Circuit, DC, R, C, GND,
  compileDslToDb,
  exportNetlist,
  importNetlist,
  reverseDbToDsl,
} from '@ssevindikx/wirescript';

// 1. Build a circuit
const original = Circuit('RC Filter', DC(5), R(1000), C(1e-6), GND());

// 2. Serialize to DB
const db = compileDslToDb(original);

// 3. Export to SPICE netlist
const spice = exportNetlist(db);
console.log(spice);
// * RC Filter
// V1 N1 0 5
// R1 N1 N2 1000
// C1 N2 0 1e-6
// .end

// 4. Import back
const rebuilt = importNetlist(spice);
console.log(rebuilt.components.length); // 3 (V, R, C)

// 5. Convert rebuilt DB back to DSL
const dsl = reverseDbToDsl(rebuilt);
console.log(dsl);
```

## CLI usage

See [CLI Reference](./cli.md) for `to-netlist` and `from-netlist` commands.

```sh
# DB JSON ŌåÆ SPICE
wirescript to-netlist circuit.json

# DSL/TS ŌåÆ SPICE (direct, no intermediate JSON needed)
wirescript to-netlist circuit.ts --format spice --out circuit.net

# DB JSON ŌåÆ WireScript CSV
wirescript to-netlist circuit.json --format ws-csv --out circuit.csv

# SPICE ŌåÆ DB JSON
wirescript from-netlist circuit.net --out circuit.json

# CSV ŌåÆ DB JSON
wirescript from-netlist circuit.csv --format ws-csv --out circuit.json
```
